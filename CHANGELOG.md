# Change Log - @paeng/playwright-ms-auth

## 1.0.0 (Initial Release)

### Features

- Flexible authentication framework supporting password and certificate authentication
- Multiple credential providers:
  - Azure KeyVault
  - Local File
  - Environment Variables
  - GitHub Secrets
- Abstract Factory pattern for extensibility
- CLI tool with full environment variable support
- Automatic storage state management with expiration
- TypeScript support with full type definitions
- Debug logging support
- Multi-region Entra endpoint support
